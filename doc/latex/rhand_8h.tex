\subsection{Detailed Description}
This is an example {\bfseries settings.\+h} for \hyperlink{servo_8ino}{servo.\+ino}.

Every Arduino board on the bot is to be a simple servo controller that accepts commands from serial input, and so they are all to share the code located in \hyperlink{servo_8ino}{servo.\+ino}. Each board needs its own header like this one to define custom information about the board and each servo on the board. This is such a settings file and will be documented as an example.
